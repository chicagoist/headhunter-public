## 🚦 КРИТИЧЕСКИЕ ПРАВИЛА ФОРМАТИРОВАНИЯ (нарушение = автоотказ)

| Правило | ✅ ПРАВИЛЬНО | ❌ НЕПРАВИЛЬНО |
|---|---|---|
| **Формат файла** | Clean PDF (text-selectable) | Word .docx, image PDF |
| **Макет** | Single-column | Multi-column, 2-3 колонки |
| **Таблицы** | НЕТ таблиц в CV | Любые таблицы |
| **Иконки/Графика** | НЕТ иконок | 📱 ✉️ иконки навыков |
| **Шрифт** | Arial, Calibri, Times New Roman | Декоративные шрифты |
| **Размер шрифта** | 10-12pt основной, 14-16pt заголовки | < 10pt |
| **Формат дат** | MM.YYYY (05.2024) | 05/2024, Mai 2024 |
| **Язык** | Немецкий (если вакансия DE) | Смешанный |
| **Длина** | 2 страницы A4 | 3+ страниц |
| **Фото** | Профессиональное фото | Без фото (в Германии ожидают) |
| **Заголовки** | Стандартные термины | Творческие/нестандартные |

## 💡 ТАКТИКИ ДЛЯ СЛОЖНЫХ СИТУАЦИЙ

### Обработка §24-статуса (Aufenthaltserlaubnis)
```
❌ НЕЛЬЗЯ: Скрывать или акцентировать отрицательно
✅ ПРАВИЛЬНО: Упомянуть в Anschreiben позитивно:

"Als zielstrebiger Fachkraft-Einwanderer mit geregeltem 
Aufenthaltstitel gemäß §18a AufenthG stehe ich dem Unternehmen 
langfristig und ohne Einschränkungen zur Verfügung."
```

### Объяснение карьерных пауз (Lücken)
```
❌ НЕЛЬЗЯ: Пробел без объяснения
✅ ПРАВИЛЬНО:
MM.JJJJ – MM.JJJJ | Weiterbildung / Selbststudium
• Erwerb von Zertifikaten: AZ-900, CCST
• Intensive Fortbildung in Java/Spring Boot und KI-Technologien
• Entwicklung von Eigenprojekten (GitHub: ...)
```

### Профессиональное фото (Bewerbungsfoto)
```
✅ Требования:
- Деловой стиль (пиджак или рубашка)
- Нейтральный/светлый фон
- Прямой взгляд в камеру
- Разрешение минимум 300 dpi
- Соотношение 3:4 (портрет)
- Современная фотография (не старше 2 лет)
```
