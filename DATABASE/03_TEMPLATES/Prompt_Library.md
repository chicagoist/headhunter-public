## 📋 ПРОМПТ №1: Диагностика и Gap-Analysis

**Назначение:** Выявить пробелы между CV и вакансией до отправки отклика

```
ПРОМПТ:
Действуй как опытный немецкий IT-рекрутер с 10 годами опыта в Германии.
Твоя задача — провести строгий gap-analysis.

МОЁ РЕЗЮМЕ:
[ВСТАВИТЬ ТЕКСТ LEBENSLAUF]

ВАКАНСИЯ:
[ВСТАВИТЬ ТЕКСТ STELLENANGEBOT]

Проведи следующий анализ:

1. SCORE: Оцени совпадение от 0 до 100 (ATS Semantic Score)
   - Tier 1 keywords (обязательные): X из Y присутствуют → N%
   - Tier 2 keywords (желательные): X из Y присутствуют → N%
   - Итоговый ATS Score: N%

2. MISSING KEYWORDS: Список отсутствующих Tier 1 ключевых слов

3. RED FLAGS: Что насторожит немецкого рекрутера?
   - Пробелы в employment history
   - Несоответствие квалификации
   - Форматные ошибки

4. TOP 3 STRENGTHS: Главные преимущества для данной позиции

5. REWRITE REQUEST: Перепиши 3 наиболее слабых bullet point 
   используя формулу: [Глагол действия] + [Технология] + [Результат] + [Метрика]

6. ANSCHREIBEN HOOK: Предложи первый абзац сопроводительного письма 
   (2-3 предложения, цепляющий, конкретный)
```

## 📝 ПРОМПТ №2: Адаптация Lebenslauf под конкретную вакансию

```
ПРОМПТ:
Ты — немецкий технический рекрутер и специалист по оптимизации резюме.
Рынок: Германия 2026. Правила: EU AI Act, DIN SPEC 91426, ISO 10667.

ЗАДАЧА: Адаптируй мой Lebenslauf под вакансию без изменения фактов.

ПРИНЦИПЫ АДАПТАЦИИ:
- Используй ТОЧНЫЕ термины из вакансии (semantic matching)
- Каждый опыт = Глагол + Технология + Результат + Цифра
- Сохраняй Human-in-the-Loop стиль (не 100% AI-generated)
- Язык: немецкий (если вакансия на немецком)
- Формат: clean, single-column, PDF-совместимый

МОЙ LEBENSLAUF (исходный):
[ВСТАВИТЬ]

STELLENANGEBOT:
[ВСТАВИТЬ]

ВЫХОДНОЙ ФОРМАТ:
1. Адаптированный раздел "Berufserfahrung" (весь)
2. Адаптированный раздел "Kenntnisse/Skills"
3. Адаптированное "Profil" / Summary (3 строки)
4. Список внесённых изменений (для контроля)
```

## ✉️ ПРОМПТ №3: Генерация Anschreiben (сопроводительного письма)

```
ПРОМПТ:
Ты — профессиональный немецкий технический писатель.
Создай Anschreiben согласно немецким стандартам 2026.

ПРОФИЛЬ КАНДИДАТА:
- Имя: [Vorname Nachname]
- Позиция: Junior Java/Spring AI Architect
- Главные навыки: Java 21, Spring Boot 3, Spring AI, Docker, Azure (AZ-900)
- Особенности: Международный опыт, §24-статус (следует упомянуть позитивно)
- Сертификаты: AZ-900 Azure, CCST Cisco, Google IT Support, Fullstack Dev

ВАКАНСИЯ:
[ВСТАВИТЬ]

КОМПАНИЯ:
[Название] — [что знаешь о компании]

ПРАВИЛА ANSCHREIBEN:
- Длина: ровно 1 страница A4
- Структура: Einleitung → Hauptteil → Schluss → Handlungsaufforderung
- Тон: профессиональный, но не слишком формальный
- НЕ ПИСАТЬ: "Hiermit bewerbe ich mich..." (клише)
- Стиль: конкретные достижения, не general claims
- Human-in-the-Loop marker: добавить 1 личную деталь о компании

ВЫХОДНОЙ ФОРМАТ:
[Полный текст Anschreiben на немецком языке]
+ [Список ключевых слов из вакансии, использованных в тексте]
```

[ВСТАВИТЬ]

ВАКАНСИЯ:
[ВСТАВИТЬ]

1. Сгенерируй 10 наиболее вероятных технических вопросов
2. Сгенерируй 5 поведенческих вопросов (Verhaltensinterviews) 
   в стиле STAR (Situation → Task → Action → Result)
3. Сгенерируй 3 каверзных вопроса о пробелах/слабых местах в CV

позитивно и профессионально (1-2 предложения)?

## 🔍 ПРОМПТ №5: Анализ компании перед откликом

```
ПРОМПТ:
Проведи разведку компании перед моей Bewerbung.

КОМПАНИЯ: [Название]
ВАКАНСИЯ: [Ссылка или описание]

Найди и структурируй:
1. PROFIL UNTERNEHMEN:
   - Основана: [год]
   - Размер: [количество сотрудников]
   - Umsatz: [оборот если известен]
   - Hauptprodukte/Dienstleistungen: [что делают]
   - Tech Stack: [технологии из открытых источников]

2. KULTUR & WERTE:
   - Tone of Voice на сайте (формальный/неформальный)
   - Erwähnte Werte (что ценят в сотрудниках)
   - Glassdoor/Kununu рейтинг и основные жалобы/похвалы

3. ATS СИСТЕМА:
   - Как называется их ATS (определить по URL карьерного сайта)
   - Как оптимизироваться под их конкретную систему

4. PERSONALISIERUNG:
   - 2-3 конкретных факта о компании для Anschreiben
   - Почему именно эта компания (аргументы)
   - Имя HR-менеджера (из LinkedIn/Impressum)
```
